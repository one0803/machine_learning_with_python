「ゼロからはじめるデータサイエンス」(Joel Grus:著, 菊池 彰:訳)の22章をもとに作成

リコメンドシステムはユーザーベースとアイテムベースに分けることが出来る。
ユーザーベースでは、対象のユーザーに類似したユーザーを探し、そのユーザーが持っている興味をおすすめする。
アイテムベースでは、対象のユーザーの持っている興味に類似した興味をおすすめする。

Amazonでのリコメンドシステムを以下に示す。上の「よく一緒に購入されている商品」がアイテムベース、
下の「この商品を買った人はこんな商品も買っています」がユーザーベースであると分かる。
